# Softora Agenda iOS

Native iPhone-app voor de Softora agenda.

## Wat deze app doet

- Inloggen met bestaande Softora premium-login.
- Aankomende agenda-afspraken ophalen via `https://www.softora.nl/api/agenda/appointments`.
- Handmatige afspraken toevoegen via `https://www.softora.nl/api/agenda/appointments/manual`.
- Dezelfde sessie-cookie gebruiken als de website, maar dan in een native iOS-ervaring.

## Openen

Open `SoftoraAgenda.xcodeproj` in Xcode.
