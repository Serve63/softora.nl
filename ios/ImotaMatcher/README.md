# IMOTA Matcher iOS Preview

Kleine iPhone-preview app voor de IMOTA Personeel Matcher HTML.

## Openen

Open `ImotaMatcher.xcodeproj` in Xcode en druk op Run met een iPhone Simulator.

## Wat deze preview doet

- Laadt `imota-matcher.html` lokaal in een iPhone WebView.
- Toont dezelfde interface als de aangeleverde HTML.
- Gebruikt demo-profielen voor de matchresultaten, zodat er geen API-key in HTML nodig is.

Deze app is bedoeld om snel te checken hoe de pagina op telefoon voelt. Voor echte AI-resultaten hoort later een veilige backend-route gebruikt te worden.
