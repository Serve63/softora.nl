# Leadlijst samenvatting

- Volledige lijst: 500 prospects met redesign-score en actief-score.
- Hot shortlist: 54 prospects met redesign-score >= 45 en actief-score >= 65.
- Warm shortlist: 340 prospects met actief-score >= 45.

Belangrijk: de actief-score is een automatische inschatting op basis van openbare websitesignalen zoals bereikbaarheid, recente jaartallen, contactgegevens, telefoonnummers, KvK/BTW/offerte/openingstijden en negatieve woorden zoals gesloten/gestopt. Dit is geen officiële KvK- of Google Maps-controle.
